# Финансоввый анализ
  

## Описание проекта

построить дашборд по финансовому анализу продаж магазина косметики, бытовой химии, предметов гигиены и т.д.


## Инструменты

- **Fastboard**
- **Python**
- **Pandas**
- **Matplotlib**
- **Seaborn**



## Общий вывод


- Разработан Дашборд [**Финансовый анализ**](<https://demo.fastboard.online/public/1660e7a3-d972-460f-87bc-d9eaba27fe7b/board/e41684f3-8c00-4ff4-ab7b-4d88a4dfba6b>) в BI-системе Fastboard.
- Проведен анализ данных и даны рекомендации.
