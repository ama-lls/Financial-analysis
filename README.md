# Финансовый анализ

## Описание проекта
- Провести обзор и проверку исходных данных с помощью Python.
- Построить дашборд по финансовому анализу продаж магазина косметики, бытовой химии, предметов гигиены и т.д. в BI-системе Fastboard.

## Инструменты

- **Fastboard**
- **Python**
- **Pandas**
- **Matplotlib**
- **Seaborn**
